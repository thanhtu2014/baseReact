# CMS Portal FE
CMS Portal FE

## Requirement

- Node version: 18
- Yarn version: 1.22.19

## Installation

- `yarn install`
- `yarn prepare`
- `cp .env.example .env`
- `yarn start`

## Production Preview

- `yarn build`
- `yarn serve`

Your app is started on port 8081

## Source Map Explorer

- `yarn build`
- `yarn analyze`
